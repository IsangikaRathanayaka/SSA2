Installation guide lines for simple gooogle drive file uploader with Oauth.
--------------------------------------------------------------------------------------------------------------------------------------------
01. Need to install Xampp setup on the linux Operating system.The sample is tested on Ubuntu x64 20.04.3 LTS
    -https://www.apachefriends.org/index.html
    
02.Install the Xampp setup on to the /Opt/lampp directory.the GUI will load for the installation.Set the listning port to "port "8080"
    # sudo ./xampp-linux-x64-8.1.5-0-installer.run
    
03.create a folder with the name "upload" in side the /opt/lampp/htdocs/
    # sudo mkdir /opt/lampp/htdocs/upload
    
04. Copy all 4 files (index.html,main.js,upload.html,upload.js) to the upload directory.

05. Start the Xampp apache service.
    #sudo /opt/lampp/xampp startapache
    
06. Now the web page can be accessed with web browser on the same server or form external address.
    - on the brwser type "http://localhost:8080/upload/
